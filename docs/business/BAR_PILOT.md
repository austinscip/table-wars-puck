## üé§ The Pitch

### Timing Your Approach

**Best time to pitch:**
- Early afternoon (2-4pm) when bar is quiet
- NOT during peak hours
- NOT on Friday/Saturday (too busy)
- Talk to owner/manager, not bartender

### The 30-Second Hook

*"Hey [Name], I'm [Your Name]. I've built this new bar game called TABLE WARS - it's like interactive electronic game pucks that people pass around the table. Think of it like a high-tech drinking game meets arcade. I'm looking for one bar to pilot it for free for 2-4 weeks. Would you be open to a 5-minute demo?"*

**Key points:**
- **FREE** for the pilot (they pay nothing)
- **Short commitment** (2-4 weeks)
- **No work for them** (you handle everything)
- **Ask for demo first** (don't oversell before showing)

### The Demo (If They Say Yes)

1. **Show the demo video** (30 seconds)
   - Pull it up on your phone
   - Let them see people playing
   - Show excitement/reactions

2. **Let them hold a puck**
   - Turn it on
   - Let them press button/shake it
   - Run through 2-3 games quickly
   - Make it interactive, not a lecture

3. **Explain the value**
   - *"People stay longer when they're having fun"*
   - *"It's Instagrammable - free marketing for you"*
   - *"Encourages group interaction, more drink orders"*
   - *"Bar vs Bar competition potential"*

4. **Address concerns preemptively**
   - **"What if someone breaks it?"** - *"They're pretty durable, and I'll swap out any broken ones immediately. I'm here to make sure it succeeds."*
   - **"What if people don't play?"** - *"That's what the pilot is for. If it doesn't work, I take them back, no hard feelings."*
   - **"How much?"** - *"Nothing for the pilot. I want your feedback. If you love it after 2-4 weeks, we can talk about a monthly fee."*

5. **Close with a specific ask**
   - *"Can I bring 6 pucks this Thursday night and set them up at one table?"*
   - *"I'll stay the whole night to explain the game and get feedback."*
   - *"Sound good?"*

### If They're Interested But Hesitant

**Offer a super low-commitment test:**
- *"How about I bring them just for one night - this Thursday from 7-11pm?"*
- *"I'll handle everything, and if people love it, we can do the full 2-week pilot."*
- *"If it flops, I take them home and we're done. No risk to you."*

### If They Say No

**Don't burn bridges:**
- *"No problem! If you hear about it working at another bar and change your mind, here's my number."*
- Leave your contact info
- Ask for referral: *"Do you know any other bar owners who might be interested?"*

## üìÖ Pilot Logistics

### Equipment You Need on Pilot Night

**The Pucks:**
- [ ] 3-6 fully charged pucks (6+ hour battery)
- [ ] Backup batteries (charged)
- [ ] Small toolkit (screwdriver for quick repairs)
- [ ] Charging cables + power bank

**Marketing Materials:**
- [ ] 2-3 laminated instruction cards
- [ ] QR code to feedback form (Google Form)
- [ ] Your business cards
- [ ] "HOW TO PLAY" poster (optional)

**Documentation:**
- [ ] Phone/camera for filming
- [ ] Notebook for feedback
- [ ] Laptop (in car as backup)

### Setup Process

**1 hour before peak time:**

1. **Claim a table**
   - Pick high-visibility table
   - Near bar (staff can see)
   - NOT hidden in corner

2. **Set up signage**
   - Instruction card on table
   - Make it look intentional, not random

3. **Test all pucks**
   - Boot each one
   - Verify games work
   - Check battery levels

4. **Brief the staff**
   - Explain it to bartenders
   - Give them simple pitch: *"It's a new game we're testing, free to play!"*
   - Tell them to send interested people your way

### During the Pilot Night

**Your role:**
- **Be present but not pushy**
- **Approach groups sitting idle:** *"Hey, want to try a new game? It's free!"*
- **Do the first demo yourself** - get people excited
- **Step back once they're playing** - let them discover it
- **Film reactions** (ask permission first!)
- **Take notes on:**
  - Which games get the most laughs?
  - Which games are confusing?
  - Do people play for 5 min or 30 min?
  - Do they order more drinks while playing?

**Red flags to watch for:**
- Pucks getting thrown/dropped
- Games too complicated (people give up)
- Not enough spectator value (boring to watch)
- Technical issues (LEDs flickering, games crashing)

**Good signs:**
- People filming it for Instagram
- Groups playing for 20+ minutes
- Bystanders asking "what is that?"
- Staff getting interested
- People asking "when is this coming back?"

## üìà Post-Pilot Analysis

### Week 1 Debrief (After First Night)

**Ask yourself:**

1. **Did people play voluntarily?**
   - Yes = Great sign!
   - No = Need better hook/marketing

2. **How long did groups play?**
   - 20+ min = Solid engagement
   - 5-10 min = Games need work
   - < 5 min = Major redesign needed

3. **Technical issues?**
   - List every bug/failure
   - Fix before next session

4. **Which games worked best?**
   - Double down on winners
   - Cut or redesign losers

5. **Bar owner's reaction?**
   - Excited = Continue pilot
   - Neutral = Show data, push harder
   - Negative = Pivot or find new bar

### Week 2-4: Iterate & Improve

**Based on feedback:**
- [ ] Update firmware (fix bugs, tweak games)
- [ ] Improve instructions
- [ ] Add requested features
- [ ] Build more pucks if demand is high
- [ ] Schedule regular nights (every Thursday?)

**Track metrics:**
- Total hours played
- Number of unique players
- Social media engagement
- Bar owner satisfaction (weekly check-in)
- Drink sales on pilot nights (ask bartender)

## üß∞ Troubleshooting Guide (During Pilot)

### Common Issues & Quick Fixes

**Puck won't turn on:**
- Swap battery with charged backup
- Check power switch
- If still dead, swap entire puck

**LEDs dim or flickering:**
- Battery dying - swap immediately
- MT3608 may need adjustment (do later)

**Game freezes/crashes:**
- Note which game caused it
- Power cycle puck (off/on)
- If repeats, disable that game mode

**Button not responding:**
- Check if it's loose
- Try gentle press vs hard press
- May need hardware fix (swap puck)

**People don't understand the game:**
- Do a live demo for next group
- Simplify instruction card
- Add visual cues to gameplay

**Puck gets dropped:**
- Check for damage immediately
- Test all functions
- If working, keep using
- Note durability for V2 design

**Bar too loud to hear instructions:**
- Rely on visual LED feedback
- Make button presses obvious
- Add bigger text to instruction card
