Start the Table Wars Flask server.

Steps:
1. Check if virtual environment exists at `server/venv`
2. If not, create it: `python3 -m venv server/venv`
3. Activate venv and install requirements: `pip install -r server/requirements.txt`
4. Start server: `python server/app.py`
5. Verify it's running on http://localhost:5001
6. Leave it running in background or inform user how to access it
